# Data Directory

This directory contains example datasets used with the NumSummarySim project.

## Files

- `sample_sales_data.csv`  
  A small synthetic dataset of daily sales and related fields, suitable for
  demonstrating numerical summaries and simulations.

## Notes

- The data is intentionally simple and small so it can be comfortably
  explored in a classroom or workshop setting.
- For real-world analysis, replace or supplement these files with your
  own datasets.
